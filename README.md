# Chmount
Run GNU/Linux distributions on Android
### By Furkan Karcıoğlu

## Installation

```
su
wget https://raw.githubusercontent.com/TheDoop/Chmount/master/chmount && chmod +x chmount
```

## Usage

```
su
chmount [path/to/chroot/folder] [command]
```

### Example

```
su
chmount /data/media/debian bash
```
